# IoT-Driven-Predictive-Maintenance-Failure-Prediction-System

**Overview**

This project builds a predictive maintenance system to identify potential machine failures using machine sensor data.

The pipeline integrates SQL, Python, Excel, and Power BI to simulate a real-world data analytics workflow.

The objective is to predict machine failure and classify failure types using machine learning models.

**Tech Stack**

SQL – Data cleaning & feature engineering

Python – Data preprocessing and machine learning

Pandas / Scikit-Learn – Data analysis and modeling

Excel – Business reporting and summary analysis

Power BI – Interactive dashboard visualization
